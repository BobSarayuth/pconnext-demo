# Security Policy

## Supported Versions

This document outlines which versions of SCG Digital AI Service (Bobby) are currently supported with security updates.

| Version | Python Version | Supported          |
| ------- | -------------- | ------------------ |
| 0.1.x   | >= 3.12, < 4.0| :white_check_mark: |
| 0.0.x   | >= 3.12, < 4.0| :x:                |

We only maintain security updates for the latest minor version of the project. Based on our pyproject.toml configuration, this project requires Python 3.12 or higher and below Python 4.0.

## Security Updates

### Version Update Policy
- **Major versions (X.0)**: Major functionality changes, new features, potential breaking changes
- **Minor versions (0.X)**: Bug fixes, security updates, and backward-compatible improvements

### Security Best Practices
- Always use the latest supported version
- Regularly update dependencies
- Follow secure configuration guidelines
- Monitor security advisories

Thank you for helping keep SCG Digital AI Service and our users secure!
